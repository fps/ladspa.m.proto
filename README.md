# ladspa.m.proto

Protobuf definitions for defining synths and instruments made up of LADSPA synths

# Requirements

* protobuf, python

# Building

type

<pre>
make
</pre>

# Installation

type

<pre>
make install
</pre>

preferrably as superuser
